## 💡 테스트 방법

### 1. 메인 페이지 확인
- `index.html` 열기
- 헤더 메뉴 동작 확인
- 주요 채용공고 로딩 확인
- 모바일 반응형 확인 (F12 → 모바일 모드)

### 2. 회원가입 테스트
- `signup.html` 열기
- 약관 동의 → 정보 입력
- @coasia.com 이메일 입력 시 내부 직원 메시지 확인
- 비밀번호 강도 체크 확인
- 회원가입 완료 페이지 확인

### 3. 로그인 테스트
- 회원가입한 계정으로 로그인
- 로그인 후 헤더 변화 확인 (사용자 메뉴 표시)
- 알림 개수 표시 확인
